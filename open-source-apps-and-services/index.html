Introduction - Why Use Open Source Software and Services?

Everyone can feel that as an individual, it has become more and more difficult to protect our privacy on the Internet, and we are also being subject to more and more non-libre restrictions. In some countries and regions, there is even a double suppression of privacy from the official and software companies, such as displaying IP address, registration requires info that can locate each individual, etc.; there are also some platforms gradually tightening the content censorship requirements, even making the daily communication restricted. However, if we want to protect our privacy and get rid of most kinds of restrictions, we cannot trust the false promises of capital and big companies. To achieve these goals, a necessary condition is: open source.

Any software installed on various electronic devices run according to a series of code written by the developer, these computer languages are called Source Code. The developer can choose not to open source, only to provide users with the software, the user can only use the functions provided by the developer, only see contents the developer wants to show, but do not know exactly what the software does (such as whether there is some collection of information involving privacy); however, the developer can also choose to open source, so that everyone can transparently know the logic of the software, and all the actions the software performed. (Note: not open source is not necessarily bad. Every developer has the right to decide how to use his or her own work results.)

With the description in the previous paragraph, it is easy to know: if the software or service is not open source, we cannot know how it’s implemented (or at least, can’t trust the developer), and thus nothing can be guaranteed. If we choose not to trust the promise of capital, then open source is a necessary condition for privacy and freedom protection. The open source software and services documented in this article were chosen mainly focusing on privacy, freedom, and encryption. (Open source also has other advantages, but we don’t go into detail here.)

Browser

  • Mull for Android
    • An open source alternative to Google Chrome.
  • LibreWolf for PC
    • An open source alternative to Microsoft Edge.
  • Tor Browser for Android and Desktop
    • Maximum protection for your privacy. Greatly recommended.
    • Sadly, in some countries and regions, Tor service is not available.

https://privacytests.org/

Chat / Communication / Social Media

  • CutTheCord
    • An F-Droid version of Discord.
    • You can log in to your Discord account and communicate with others just like what you do in the Discord App.
    • But please note that this is a frontend client. The Discord service itself is non-libre. Just using the Discord service, despite using this client, won’t fully protect your privacy. (Trivia: Tencent heavily invests in Discord)
    • In some countries and regions, Discord service is not available.
  • Fritter
    • An open source client for browsing Twitter.
    • You can view other users’ tweets through this client without logging in. Also you can set up groups to follow to receive update notifications. Of course, you can’t like, comment or retweet.
    • In some countries and regions, Twitter service is not available.
  • K-9 Mail
    • An open source alternative to Mail apps.
    • Not only support all normal email sending, receiving and management operations, but also can send and receive encrypted or signed emails with OpenKeyChain, which will be mentioned in the followings.
  • Molly
    • An F-Droid version of Signal.
    • You can log in to your Signal account and communicate with others just like what you do in the Signal App.
    • But please note that this is a frontend client. The Signal service itself, while supporting end-to-end encryption, still has many non-libre elements.
      • Signal isn’t fully open source.
      • Signal service isn’t fully anonymous, since it requires your phone number to register.
      • Signal service is not decentralized. So it can be blocked easily by some governments.
    • In some countries and regions, Signal service is not available.
  • NewPipe
    • An open source client for browsing YouTube.
    • You can browse YouTube videos through this client without logging in. Also you can set up subscription groups to receive update notifications. Of course, you can’t like or comment.
    • In some countries and regions, YouTube service is not available.
  • Session
    • An open source alternative to instant messenger like WhatsApp / Facebook Messenger.
    • Support text, picture and voice messages.
    • Its advantages over other instant messengers:
      • Fully anonymous to register.
      • Run on decentralized servers.
      • Communicate in a Tor-like way to hide your IP.
      • End to end encrypted.
      • Support message disappearing timer.
  • Tusky / Husky / FediLab Lite
    • Mastodon and Pleroma are open source alternatives to social medias like Twitter / Facebook.
    • Open source clients of open source social platform protocol Mastodon / Pleroma.
    • Eliminate capital’s interference in social media pushing - use open, transparent pushing algorithms. Escape the information cocoon and refuse to be what they want us to be.

Connectivity

  • Orbot

    • Can hide the IP through the Tor service while using any software.

    • Sadly, in some countries and regions, Tor service is not available.

  • Riseup VPN

    • A VPN software. The point of using a VPN is not only to break through some walls, but also to protect privacy and enhance your security.

    • Sadly, in some countries and regions, Riseup VPN is not available.

  • WireGuard

    • A protocol and software that is super easy to build VPN tunnels. If conditions allow, you can use it to build a VPN under legal conditions.

File Manager

  • Material Files
    • An open source alternative to File Manager apps.
    • Support SFTP and SMB servers.

Search Engine

  • Duckduckgo
    • An open source alternative to search engines like Google / Bing.
    • Sadly, in some countries and regions, DuckDuckGo service is not available.
  • MetaGer
    • An open source alternative to search engines like Google / Bing.

Security

  • OpenKeyChain
    • A PGP key generator and manager that can be used with K-9 Mail (mentioned above) to send encrypted or signed emails.
  • KeePassDX
    • A password manager that can record, auto-fill passwords and generate strong passwords.
  • KeePassXC for PC
    • Same as KeePassDX, but on desktop.

Sync

  • Nextcloud
    • An open source alternative to Cloud services.
    • If conditions allow, it is best to set up a self-hosted cloud drive through the Nextcloud service. If not, Nextcloud also offers a number of cloud service providers that offer their services for free.
  • Syncthing
    • Synchronizes files in a similar way to BitTorrent.

Time-management

  • Etar

    • An open source alternative to Calendar apps.

    • Support Etesync service to sync, which can be self-hosted.

  • Simpletask

    • An open source alternative to task managers like Microsoft ToDo.
    • Based on todo.txt, easy to sync and edit in formatted way.

  • TimeLimit.io

    • An open source alternative to Lock your phone.
    • All locking software involves high-risk Device Manager permissions. Please read the documentation carefully before using and decide for yourself whether to use it. The author of this article is not responsible for any problems with your device caused by using this software.

Tools

  • AnkiDroid
    • A Flashcard application, especially useful for memorization purposes (e.g., word memorization).
  • Apk Extractor
    • Extract apk file from installed apps.
  • Metronome
    • A simple metronome.
  • MoneyWallet
    • An open source alternative to Money Wallet apps.
    • Support export as json / csv.
  • PDF Converter
    • Create pdf from images, and support some simple operations like greyscale or compression.
  • Phyphox
    • Get data from sensors of your phone, like air pressure or acceleration.
  • Track & Graph
    • Input data and draw charts
  • WebApps
    • Many websites support Webapps. You can open a website as webapp using this app.

Writing / Collaboration

  • AnySoftKeyboard
    • An open source alternative to keyboard / input method like Google IME.
    • Support multi-lingual.
    • If you use open-source instant messenger but still use IME from big companies, they will still know everything you input, and thus there is no privacy. It’s very important to use open source IME!
  • cryptpad.fr (website)
    • Can be used as an open source alternative to some collaboration suite.
  • Easy Diary
    • An open source alternative to Diary apps.
    • Support export as xls.
  • MarkText (Desktop)
    • An open source alternative to markdown editors like Typora.
  • Quillnote
    • An open source alternative to Note apps.
    • A note app using Markdown. Support sync through built-in Nextcloud Sync service.

引言 - 为什么使用开源软件和服务

相信大家都能感觉到，作为一个普通的个体，在网络上已经越来越难以保护好自己的隐私，也在受到越来越多的不自由的限制。在一些国家和地区，存在着来自官方和软件公司的对隐私的双重打压，比如显示IP地址、注册必须填写可以定位到个人的信息等等；也有一些平台正在渐渐收紧内容检查的要求，甚至使得日常交流都受到限制。然而，如果想保护好自己的隐私、想脱离各种限制，我们不能信任资本、大公司的假意的承诺。想达成这些目标，一个必要条件就是：开源。

开源，是开放源代码的简称。安装在各种电子设备上的软件，都是按照开发者编写的一系列计算机指令运行的，这些计算机指令就叫做源代码(Source Code)。开发者可以选择不公开源代码，只为提供使用者提供代码编译生成的软件，这时使用者只能使用开发者提供的功能，只会看到开发者想展示的内容，但不知道软件具体到底做了什么（比如有没有搜集涉及隐私的资料）；同样地，开发者也可以选择开放源代码，这样所有人都可以透明地看到软件运行的逻辑、软件执行的所有行为。（_但请注意，不开源不一定是罪恶的，每一位开发者都有决定自己的成果的使用方法的权利。_）

通过上一段的描述，我们很容易知道：如果软件或服务闭源，那我们就看不见它的实现逻辑，从而一切都无从谈起。如果我们选择不信任资本的承诺，那么开源是隐私和自由保护的必要条件。这篇文章记录的开源软件和服务，主要的选择依据就是注重隐私、自由、加密。（开放源代码也有更多的优点，在这里我们暂且不展开。）

正文 - 开源软件和服务介绍

在往下阅读之前，您也可以先参考这篇文章，在我学习开源的路上很多地方都受到它的启发：

Favorite apps and services essays

接下来的内容分类是按照拼音首字母排序的。

安全

  • OpenKeyChain
    • 一款PGP密钥生成和管理器，可以和 K-9 Mail 配合使用来发送加密邮件。
  • KeePassDX
    • 一款密码管理器，可以记录和自动填入密码，也可以生成强密码。
  • KeePassXC （桌面版）
    • 一款密码管理器，同 KeePassDX。

记录和协作

  • cryptpad.fr （网站）
    • 可以作为一些较简单的协作服务（如 腾讯收集表）的开源替代
    • 一个协作网站，可以执行文本、简单的 code 、收集表等的多人在线协作。
  • Easy Diary
    • 可以作为 日记 软件的开源替代。
    • 一款日记软件，支持导出为xls格式。
  • MarkText （桌面版）
    • 可以作为 Typora 等 Markdown 编辑器的开源替代。
  • Quillnote
    • 可以作为 便签 软件的开源替代。
    • 一款使用 Markdown 格式的便签软件。内置 Nextcloud （后面会提到，是自由的、支持自托管(self-host) 的云盘服务）同步。
  • Trime (桌面版对应 rime)
    • 可以作为 搜狗输入法 等输入法的开源替代。
    • 一款高度可自定义的中文开源输入法。
    • 尽管使用再强的加密通信软件，如果依然使用不开源的输入法，那么大公司都可以记录你输入的所有内容（包括密码），隐私都无从谈起。所以，使用开源输入法十分重要。

聊天、通信、媒体

  • CutTheCord
    • 是 Discord 软件的一个 F-Droid 版本。
    • 你可以像在 Discord App 中一样，登录你的 Discord 账号、和他人进行交流。
    • 但请注意，此应用是一个前端客户端，而Discord 服务本身是不自由的。只要使用 Discord 服务，尽管使用这个客户端，也不能完全保护你的隐私。（小知识：腾讯是 Discord 的一个大力投资者）
    • Discord 服务在一些国家和地区不能直接使用。
  • Fritter
    • 是浏览 Twitter 的一个开源客户端。
    • 你可以在不登录的情况下通过这个客户端浏览其他用户的推文，你还可以设置关注组来收到更新通知。当然，你不能点赞、评论或转推。
    • Twitter 服务在一些国家和地区不能直接使用。
  • K-9 Mail
    • 是 邮件 应用的开源替代。
    • 它除了支持所有平常的邮件收发和管理操作，还可以配合上面提到的 OpenKeyChain 收发加密、签名邮件。
  • Molly
    • 是 Signal 软件的一个 F-Droid 版本。
    • 你可以像在 Signal App 中一样，登录你的 Signal 账号、和他人进行交流。
    • 但请注意， 此应用是一个前端客户端。Signal 服务本身虽然支持端对端加密，但仍然存在很多不自由因素：
      • Signal 服务不是完全开源的；
      • Signal 服务不是完全匿名的：你需要一个有效的电话号码来注册账号；
      • Signal 服务不是去中心化的。一些政府可以轻松地屏蔽其服务。
    • Signal 服务在一些国家和地区不能直接使用。
  • NewPipe
    • 是浏览 YouTube 的一个开源客户端。
    • 你可以在不登录的情况下通过这个客户端浏览 YouTube 视频，你还可以设置订阅组来收到更新通知。当然，你不能点赞、评论。
    • YouTube 服务在一些国家和地区不能直接使用。
  • Session
    • 是 腾讯QQ、微信 等即时通信软件的开源替代。
    • 它支持文本、图片和语音消息交流。
    • 它相对于其他即时通信软件的优点：
      • 注册是完全匿名的；
      • 在去中心化服务器上运行；
      • 通过类似 Tor 的方式传输数据以隐藏 IP；
      • 传输内容端对端加密；
      • 支持消息定时销毁。
  • Tusky / Husky / FediLab Lite
    • 是 新浪微博、QQ空间 等社交媒体平台的开源替代。
    • 是开源的社交平台协议 Mastodon / Pleroma 的一些开源客户端。
    • 消除了资本对社交媒体推送的干预，使用公开、透明的推送算法。逃离信息茧房，拒绝做资本想让我们成为的样子。

浏览器

  • Mull
    • 是 夸克浏览器 等浏览器的一款注重隐私保护的开源替代。
  • LibreWolf （桌面版）
    • 是 Edge浏览器 等浏览器的一款注重隐私保护的开源替代。
  • Tor Browser （安卓和桌面版）
    • 可以极大地隐藏你的身份。
    • 遗憾的是，Tor 服务在一些国家和地区不能直接使用。

https://privacytests.org/

下面的网站可以测试你现在使用的浏览器的隐私保护能力：

https://coveryourtracks.eff.org/

时间管理

  • Etar
    • 是 日历、日程表 软件的开源替代。
    • 支持可 self-host 的 Etesync service 同步。
  • Simpletask
    • 是 todo、待办 等任务管理软件的开源替代。
    • 基于 todo.txt，易于同步、格式化编辑。
  • TimeLimit.io
    • 是 锁机 软件的开源替代。
    • 所有锁机软件都涉及高风险的设备管理器权限。请在使用前仔细阅读说明文档，自行决定是否使用。使用此软件导致设备出现任何问题，本文作者不负任何责任。

搜索引擎

  • Duckduckgo
    • 是 百度、必应 等搜索引擎的开源替代。
    • 是目前最好用的开源、不收集用户数据的搜索引擎之一。
    • 遗憾的是，DuckDuckGo 服务在一些国家和地区不能直接使用。
  • MetaGer
    • 是 百度、必应 等搜索引擎的开源替代。
    • 一些由于条件限制不能使用 DuckDuckGo 的用户可以尝试使用。

同步

  • Nextcloud
    • 是 百度网盘 等云存储服务的开源替代。
    • 如果有条件，最好通过 Nextcloud 服务搭建 self-host 的网盘。如果没有对应的条件，Nextcloud也提供了一些免费提供服务的云服务提供商。
  • Syncthing
    • 它通过类似 BitTorrent 的方式进行文件同步。

网络连接

  • Orbot
    • 可以在使用任何软件时通过 Tor 服务隐藏 IP。
    • 遗憾的是，Tor 服务在一些国家和地区不能直接使用。
  • Riseup VPN
    • 一款 VPN 软件。使用 VPN 的意义不仅是突破一些受限制访问内容，还可以保护隐私、增强你的安全性。
    • 遗憾的是，Riseup VPN 在一些国家和地区不能直接使用。
  • WireGuard
    • 一款可以方便地搭建VPN隧道的协议、软件。如果有条件，可以通过它在合法的前提下搭建 VPN。

文件管理

  • Material Files
    • 是 文件管理 软件的开源替代。
    • 特色功能是支持 SFTP 和 SMB 服务器。

小工具

  • AnkiDroid
    • 是一款 Flashcard 应用，尤其是一些记忆性（如：背单词）用途上很有用。
  • Apk Extractor
    • 可以导出已安装应用的安装包。
  • Metronome
    • 一款简单的节拍器。
  • MoneyWallet
    • 是 钱包、记账 软件的开源替代。
    • 支持导出 json, csv 等多种常见格式。
  • PDF Converter
    • 可以导入图片生成 pdf，也可以做简单的（灰度、压缩等）pdf 操作。
  • Phyphox
    • 可以获取一些手机传感器收集的数据。如当前气压、加速度等。
  • Track & Graph
    • 可以输入数据，输出图表。
  • WebApps
    • 一些网页是支持 Webapp 的。可以将网页作为 Webapp 打开。
